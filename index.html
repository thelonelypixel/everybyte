Buy why?

I wanted to build this website as a challenge to myself — to see just how small a footprint a modern website could potentially have. Every single aspect of this project has been carefully considered with sustainability in mind.

☀️ The site is powered entirely by a solar panel and battery, running on a Raspberry Pi Zero mounted on a roof.
✌️ It’s built with hand-coded HTML and CSS — no frameworks, no tracking, no analytics, no unnecessary JavaScript.
🔨 Even the code editor and tools I used to write and deploy the site were chosen for their minimal energy impact.
